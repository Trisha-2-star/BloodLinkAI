# BloodLink AI – Smart Blood Donor Matching System

## Overview

BloodLink AI is a web-based application designed to help patients quickly find suitable blood donors during emergencies. The system enables users to search for donors based on blood group and city while prioritizing urgent cases. It also includes a simple AI assistant to guide users with blood donation-related queries.

## Features

* Blood donor matching based on blood group and city
* Emergency priority handling for urgent cases
* Dashboard displaying donor statistics
* AI assistant for user guidance
* Simple and user-friendly interface

## Tech Stack

* HTML
* CSS
* JavaScript

## Project Workflow

1. Users enter patient details.
2. Select the required blood group and city.
3. Choose the emergency priority level.
4. Click the **Find Donor** button.
5. The system filters donor data and displays matching donors.
6. High-priority cases activate emergency routing.
7. Users can interact with the AI assistant for blood donation queries.

## Challenges and Solutions

One of the key challenges was efficiently matching donors using multiple conditions such as blood group and city. This was solved using JavaScript's `filter()` method to retrieve and display only relevant donor information instantly without reloading the page.
